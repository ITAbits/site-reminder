# Site Reminder
Simple discord bot to remember some members of ITAbits to work on our website.

## Build

- Clone the project. 

- Install dependencies:
```
npm install
```

- Set the enviroment variables

```
touch .env
vim .env
```
And write `BOT_TOKEN=(YOUR_BOT_TOKEN_GOES_HERE)`in .env


- Run the bot

```
node bot.js
```
